# KB: Tools

## What belongs here

- CLI utility issues and usage patterns
- Script debugging (`bash`, `python`, `awk`, `jq`)
- Development tooling (VS Code, SSH, pytest, pip)
- Monitoring and profiling tools (`htop`, `iotop`, `perf`)
- Custom scripts in `scripts/`

## Naming

`YYYY-MM-DD_short_snake_case_topic.md`

## What does NOT belong here

- OS-level configuration (use `linux/`)
- Git-specific issues (use `git/`)
- Docker tooling (use `docker/`)
