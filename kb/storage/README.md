# KB: Storage

## What belongs here

- Disk detection and partitioning (`lsblk`, `fdisk`, `sfdisk`, `parted`)
- Filesystem creation and repair (`mkfs`, `fsck`, `e2fsck`)
- Mount/unmount issues and fstab configuration
- SMART monitoring and disk health
- NAS provisioning flow issues (`usb_share.py`)
- Atomic write / durability issues
- NVMe, SATA, and USB storage

## Naming

`YYYY-MM-DD_short_snake_case_topic.md`

## What does NOT belong here

- USB hotplug/detection (use `usb/`)
- SMB share configuration (use `smb/`)
- File permission issues (use `permissions/`)
