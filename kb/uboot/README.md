# KB: U-Boot

## What belongs here

- U-Boot environment variables and configuration
- Boot sequence issues (SPL, TPL, ATF)
- Serial console debugging
- Boot media selection (eMMC, SD, NVMe, USB)
- Device tree loading from U-Boot

## Naming

`YYYY-MM-DD_short_snake_case_topic.md`

## What does NOT belong here

- Kernel boot issues after handoff (use `kernel/`)
- OS-level boot issues (use `linux/`)
