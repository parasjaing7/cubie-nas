# KB: USB

## What belongs here

- USB device detection and enumeration
- Hotplug and udev rules
- USB storage mount/unmount
- USB power management and port resets
- USB3/USB2 compatibility issues
- Device node issues (`/dev/sd*`)

## Naming

`YYYY-MM-DD_short_snake_case_topic.md`

## What does NOT belong here

- Filesystem or partition issues (use `storage/`)
- Kernel driver issues (use `kernel/`)
- General mount/fstab issues (use `storage/`)
