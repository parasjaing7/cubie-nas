# KB: Networking

## What belongs here

- IP address configuration (static, DHCP, nmcli, interfaces.d)
- DNS resolution issues
- Firewall rules (ufw, iptables, nftables)
- Routing and gateway configuration
- Network interface issues (link down, driver, ethtool)
- Cubie NAS `apply_network_config()` issues

## Naming

`YYYY-MM-DD_short_snake_case_topic.md`

## What does NOT belong here

- SMB/NFS protocol issues (use `smb/` or `services/`)
- Kernel network driver issues (use `kernel/`)
- Docker networking (use `docker/`)
