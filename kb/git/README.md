# KB: Git

## What belongs here

- Git workflow issues (merge, rebase, conflict resolution)
- Deploy sync issues (rsync, .env preservation)
- Branch management and tagging
- CI/CD pipeline issues
- `.gitignore` and tracked file problems

## Naming

`YYYY-MM-DD_short_snake_case_topic.md`

## What does NOT belong here

- Code logic bugs (use the relevant service category)
- Docker deployment (use `docker/`)
