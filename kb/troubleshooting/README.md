# KB: Troubleshooting

## What belongs here

- Cross-cutting issues that span multiple categories
- Complex multi-symptom failures
- Issues where root cause was initially unclear
- Post-mortem analyses

## Naming

`YYYY-MM-DD_short_snake_case_topic.md`

## What does NOT belong here

- Issues that clearly belong in a single category (route to the correct folder)
- Speculative investigations without a confirmed root cause
