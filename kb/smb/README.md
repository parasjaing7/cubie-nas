# KB: SMB / Samba

## What belongs here

- Samba configuration issues (`smb.conf` syntax, sections, parameters)
- Share access problems (authentication, guest, valid users)
- `testparm` validation failures
- Cross-platform SMB client issues (Windows, macOS, Linux)
- Performance tuning for SMB shares

## Naming

`YYYY-MM-DD_short_snake_case_topic.md`

## What does NOT belong here

- File permission issues unrelated to Samba (use `permissions/`)
- Network connectivity issues (use `networking/`)
- Disk/mount issues (use `storage/`)
