# KB: Services

## What belongs here

- systemd unit issues (start, stop, enable, mask, dependencies)
- Cubie NAS service (`cubie-nas.service`) lifecycle
- Samba (`smbd`), NFS, SSH, FTP service management
- Service sandboxing and resource limits
- Uvicorn / FastAPI runtime issues

## Naming

`YYYY-MM-DD_short_snake_case_topic.md`

## What does NOT belong here

- Samba config file issues (use `smb/`)
- Docker container lifecycle (use `docker/`)
- Networking issues (use `networking/`)
