# KB: Kernel

## What belongs here

- Kernel panics and oops analysis
- Module loading issues (`modprobe`, `lsmod`, `dmesg`)
- Device tree overlays and DTS issues
- Driver compatibility problems
- Kernel version-specific regressions

## Naming

`YYYY-MM-DD_short_snake_case_topic.md`

## What does NOT belong here

- U-Boot / bootloader issues (use `uboot/`)
- Userspace service issues (use `services/` or `linux/`)
- USB userspace issues (use `usb/`)
