# KB: Linux

## What belongs here

- OS-level issues: systemd, apt/dpkg, boot sequence, journalctl, cron, PAM
- Debian/Ubuntu-specific fixes
- Log analysis patterns
- System resource issues (OOM, disk full, inode exhaustion)

## Naming

`YYYY-MM-DD_short_snake_case_topic.md`

## What does NOT belong here

- Kernel-level issues (use `kernel/`)
- Network configuration (use `networking/`)
- Service-specific issues (use `services/`)
- Docker issues (use `docker/`)
