# KB: Docker

## What belongs here

- Container lifecycle (start, stop, restart, logs)
- Image management (pull, build, prune)
- Docker networking (bridge, host, port mapping)
- Volume and bind mount issues
- Docker Compose configuration
- Resource limits and OOM

## Naming

`YYYY-MM-DD_short_snake_case_topic.md`

## What does NOT belong here

- Host OS issues (use `linux/`)
- Host networking (use `networking/`)
- NAS storage provisioning (use `storage/`)
